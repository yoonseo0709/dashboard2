# 데이터와 시각화 결과 설명

< 서울시 도시 데이터 분석>

1. 분석 목적: 
서울시 행정동/자치구 단위의 부동산 거래, 대중교통 이용, 공공자전거(따릉이) 인프라 데이터를 통합 분석하여 지역별 주거 환경의 특성과 가성비 지역 도출


2. 사용 데이터
총 4개의 테이블로 구성된 DB를 사용

1) 부동산
- 컬럼: 자치구코드, 자치구_명칭, 법정동명, 물건금액(만원), 건축년도, 건물용도 등
- 설명: 서울시 오피스텔, 연립다세대 실거래가 정보

2) 대중교통
- 컬럼: 기준_날짜, 행정동_ID, 승객_수, 00시~23시
- 설명: 행정동별 일자, 시간대별 대중교통 승하차 인원

3) 따릉이 
- 컬럼: 대여소_ID, 자치구_명칭, 위도, 경도
- 설명: 서울시 공공자전거 대여소 위치 현황

4) 행정동_마스터
- 컬럼: 행정동_ID, 행정동_명칭, 자치구_명칭, 시도_명칭
- 설명: 행정동, 자치구 매핑 기준 테이블


3. 데이터 전처리

1) 컬럼명 통일
테이블 간 조인 기준 컬럼의 명칭이 달라 JOIN조건이 복잡해지는 문제 방지하고자, 동알한 의미를 가진 컬럼은 이름을 통일했다. 예를 들어 자치구는 나타내는 컬럼은 모든 테이블에서 "자치구_명칭"으로 맞췄다.

2) 불필요한 컬럼 제거 
분석에 직접적으로 사용되지 않는 컬럼은 삭제하여 데이터 구조를 단순화했다.

3) 따릉이 주소 파싱 - 자치구 추출
따릉이 테이블의 주소 컬럼은 "서울특별시 중랑구 용마산로 644"와 같이 전체 주소 형태로 저장되어 있었다. 이 상태로는 자치구 단위 조인이 불가능하기 때문에, 텍스트 나누기(공백 기준 분리)를 통해 두 번째 토큰인 자치구명만 추출한 뒤 컬럼명을 "자치구_명칭"으로 변경했다. 이를 통해 부동산, 행정동_마스터 테이블과 동일한 키로 조인할 수 있는 형태로 만들었다.


4. 분석 결과

1) 자치구별 평균 부동산 거래금액(Top15)
강남구, 용산구, 서초구가 상위 3위를 차지하며 나머지 자치구와 큰 차이를 보였다. 
상위 15개 중 하위권을 차지한 성북구, 동대문구, 중구와는 약 2배의 차이를 보여주었다. 
이를 통해 서울 내 부동산 양극화 문제를 볼 수 있다. 

2) 대중교통 이용량 vs 부동산 가격 산점도
교통 이용량과 부동산 가격의 관계는 자치구마다 다르게 나타난다.
관악구는 교통 접근성이 좋은 지역임에도 평균 거래가가 비교적 낮아, 교통 인프라 대비 저평가 지역으로 보인다.
반면 용산구는 승객 수가 비교적 많지 않음에도 거래가가 매우 높은 것을 볼 수 있으며, 강남구의 경우 승객 수와 거래가 모두 가장 높은 것을 볼 수 있다.

3) 따릉이 인프라 대비 평균 부동산 가격
따릉이 대여소 수가 가장 많은 자치구는 송파구로, 평균 거래가는 강남,용산,서초보다 낮다.
거래가가 상위 3개 자치구에 비해 낮으면서 따릉이 인프라가 매우 잘 되어있다는 점에서 실용적인 주거지로 보인다.

전체적으로 대여소 수가 많다고 해서 거래가가 높은 것은 아니다. 
따릉이는 고가 지역에 집중되어 있는 것은 아닌  서울 전역에 균등하게 보급된 인프라임을 볼 수 있다. 
즉 따릉이 밀도는 부동산 가격과 독립적인 변수로, 주거지 선정 시 보조 편의 지표로 활용하는 것이 적합하다.

4) 주요 자치구별 시간대별 대중교통 이용 히트맵
상위 10개 자치구 모두 저녁 시간대에 대중교통 이용이 가장 많은 것을 볼 수 있다.
저녁 시간대에는 강남구와 중구에 승객수가 가장 많은 것을 볼 수 있으며 심야 시간대에는 승객수가 현저히 줄어드는 것을 볼 수 있다.
